Get started now !

Download the node for your operating system, and run your Nodix fullnode with wallet and block explorer !

Nodix win 32/64 bits
Nodix linux 32/64 bits
Run launcher.exe for windows, or launcher for linux.

The node should start synchronizing

Open the following address "http://localhost:16820/" in your web browser to access the local node interface.

The block explorer is located at this address "http://localhost:16820/nodix.site/blocks"

The html5 wallet is located at this address "http://localhost:16820/nodix.site/wallet"
