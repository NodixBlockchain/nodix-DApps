The project

Nodix engine incorporate unique object oriented language inspired from emerald language, which provide performant and distributed dynamic object manipulation.

The blockchain engine allow to store object definition, application data, files, layouts, modules and scripts in a tree of transaction on the blockchain.

The framework provide multi thread lockless access to dynamic object hierarchy, to build easily real time computationally intensive distributed application, exposed via JSON/RPC interface, as well as to generate HTML5 WebApps using the script engine to share live variables with javascript application with asynchronous parrallized access.

Fully featured blockchain node with sha256 or scrypt POW, as well staking version 1,2 and 3 implemention is available for download here, source code and build instruction for windows and linux on github.

Features

scalability

  • Distribute application data, layouts, and code on the blockchain.

  • Lockless multithread architecture for maximum multi core efficiency.

  • Easy access to data from the network into HTML5 application.

  • Portable binary modules for distributed processing.

security

  • HTML5 wallet with in browser cryptography.

  • Secure C runtime with garbage collection.

  • Signed Application data stored on blockchain.

  • Diffie Hellman key exchange for private data.

Customisation

  • Blockchain parameters defined via simple configuration file and script.

  • Fully featured HTTP application server with JSON/RPC.

  • Unique script language to generate HTML5/JS webapp front end.

  • Create new application via HTML5 user interface.

Perfomance

  • Low memory footprint, Works on 32 bits XP with 512Mo of RAM.

  • Portable binary module suporting SIMD and SSE.

  • Full runtime in C with efficient memory management.

  • Lockless access to global data shared between threads / cores.
