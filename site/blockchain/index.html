Node definition

Nodix framework is designed around event programing model, using event object to safely pass data across different modules and script asynchronously, allowing to easily program event handler for network messages as script routines, and use all the power of the binary modules in C to process variables arrays and objects manipulated in the scripts.

The script language is documented in details in the framework section.

For node definition script, it contain one object to define initial configuration for the blockchain, a node object definition, and events handlers for network messages. Messages are pre parsed by the protocol module from binary data and dispatched to the defined handling routine.
